Prompt: text

Asks the user for a standard string input with supporting validation filters and default fallbacks.

project_name = tinte.text(
    message="What is the project name?",
    placeholder="my-cool-app",
    validate=lambda value: "Name cannot be empty" if not value.strip() else None
)
name, err := tinte.Text(tinte.TextOptions{
    Message:     "What is the project name?",
    Placeholder: "my-cool-app",
    Validate: func(s string) string {
        if len(s) == 0 {
            return "Name cannot be empty"
        }
        return ""
    },
})

Parameters / Fields Specification

Parameter / Field Type Description
message / Message string The query text displayed to the user.
placeholder / Placeholder string Faint fallback text displayed when input is empty.
validate / Validate callback function A validator returning an error message string (or None/empty string if input is valid).

Prompt: password

Masks user inputs with character hiding. Ideal for API tokens and database credentials.

token = tinte.password(
    message="Enter your database password:",
    placeholder="Min 6 characters",
    validate=lambda v: "Too short" if len(v) < 6 else None
)
token, err := tinte.Password(tinte.PasswordOptions{
    Message:     "Enter your database password:",
    Placeholder: "Min 6 characters",
    Validate: func(s string) string {
        if len(s) < 6 {
            return "Too short"
        }
        return ""
    },
})

Prompt: select

A single-option dropdown prompt supporting description hints for option tags.

choice = tinte.select(
    message="Choose a frontend framework:",
    options=[
        tinte.Option("react", "React", hint="SPA library"),
        tinte.Option("vue", "Vue", hint="Progressive framework")
      ]
)
choice, err := tinte.Select(tinte.SelectOptions{
    Message: "Choose a frontend framework:",
    Options: []tinte.Option{
        {Value: "react", Label: "React", Hint: "SPA library"},
        {Value: "vue", Label: "Vue", Hint: "Progressive framework"},
    },
})

Display: table

Outputs structured rows of aligned data with CJK double-width spacing support and 4 distinct border shapes.

tinte.table(
    headers=["Dependency", "Status"],
    rows=[
        ["colorama", "Installed"],
        ["wcwidth", "Installed"]
    ],
    border="rounded", # rounded, sharp, double, minimal
    align={"Dependency": "left", "Status": "right"}
)
tinte.Table(
    []string{"Dependency", "Status"},
    [][]string{
        {"colorama", "Installed"},
        {"wcwidth", "Installed"},
    },
    "rounded", // rounded, sharp, double, minimal
    map[string]string{"Dependency": "left", "Status": "right"},
    true,      // showRowSeparators
)

Display: tree

Displays nested dictionaries, JSON, or maps. When interactive is enabled, users can expand/collapse tree folders in real-time in their console.

tinte.tree(
    data={
        "server": {"port": 8080, "host": "127.0.0.1"},
        "debug": True
    },
    title="config.json",
    expand_depth=2,
    interactive=True
)
tinte.Tree(
    map[string]interface{}{
        "server": map[string]interface{}{"port": 8080, "host": "127.0.0.1"},
        "debug": true,
    },
    "config.json",
    2,    // expandDepth
    true, // interactive
)

Theme Properties

Define a custom Theme structure using Hex color hashes. Below is the list of properties governing how color targets map to the CLI UI elements.

Theme Property Styling Scope
Primary Focus elements, active symbols, prompt prefixes, and headers.
Secondary Option hints, active checkbox tags, and minor focus lines.
Success Checkmarks, positive states, and diff additions.
Error Crosses, invalid validation messages, and diff deletions.
Muted Faint vertical guidelines (│), placeholders, and descriptions.

Custom SymbolSet

Change the default Unicode characters representing lines, bullets, checkboxes, and frames.

custom_symbols = tinte.SymbolSet(
    line_vertical="┃", # Bold vertical line
    step_active="◆",
    check_active="[x]",
    check_inactive="[ ]"
)
my_theme = tinte.Theme(
    name="BoldTheme",
    primary="#B4A7D6",
    symbols=custom_symbols
)
tinte.set_theme(my_theme)
customSymbols := tinte.SymbolSet{
    LineVertical:  "┃",
    StepActive:    "◆",
    CheckActive:   "[x]",
    CheckInactive: "[ ]",
}
myTheme := tinte.Theme{
    Name:    "BoldTheme",
    Primary: "#B4A7D6",
    Symbols: customSymbols,
}
tinte.SetTheme(myTheme)
