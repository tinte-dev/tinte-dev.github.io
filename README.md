# tinte-docs 🎨

This repository hosts the official landing page and documentation website for the `tinte` project, an elegant, cohesive terminal prompt and display library for Python and Go.

---

## Structure

* **`index.html`**: The main landing page with interactive scroll-animated terminal simulations and translation systems.
* **`docs/`**: Markdown source documentation.
  - `introduction.md`: Core reasons to adopt `tinte` and comparisons with other libraries.
  - `developer_guide.md`: Complete Python/Go API documentation.
  - `repository_separation_plan.md`: The repository splitting design guide.

---

## Web Hosting

This website is configured to be hosted on **GitHub Pages**.

1. Go to **Settings** -> **Pages** in this GitHub repository.
2. Under **Build and deployment**, set the source to **Deploy from a branch**.
3. Select **`main`** branch and **`/ (root)`** folder, then click Save.
4. The site will be available at `https://tinte-dev.github.io` (or your configured custom domain).
